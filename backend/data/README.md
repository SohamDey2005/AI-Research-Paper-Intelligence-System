# Dataset Folder

This folder stores the dataset and generated files required for semantic search.

These files are **NOT included** in the GitHub repository because they are too large for GitHub's file size limit.

## Required Files

After setup, this folder will contain:

```
backend/data/
│
├── arxiv-metadata-oai-snapshot.json
├── ai_ml_papers.csv
├── paper_embeddings.npy
└── paper_faiss.index
```

---

## Step 1: Download the Dataset

Download the **arXiv Metadata Dataset** from one of the following sources:

- Kaggle: https://www.kaggle.com/datasets/Cornell-University/arxiv
- Official arXiv Metadata: https://www.kaggle.com/datasets/Cornell-University/arxiv

Rename the downloaded file as:

```
arxiv-metadata-oai-snapshot.json
```

Place it inside:

```
backend/data/
```

---

## Step 2: Filter AI Research Papers

From the **backend** directory, run:

```bash
python scripts/filter_dataset.py
```

This creates:

```
backend/data/ai_ml_papers.csv
```

---

## Step 3: Generate Sentence Embeddings

Run:

```bash
python scripts/generate_embeddings.py
```

This creates:

```
backend/data/paper_embeddings.npy
```

---

## Step 4: Build the FAISS Index

Run:

```bash
python scripts/build_faiss.py
```

This creates:

```
backend/data/paper_faiss.index
```

---

## Final Folder Structure

After completing all the steps, your folder should look like:

```
backend/data/
│
├── arxiv-metadata-oai-snapshot.json
├── ai_ml_papers.csv
├── paper_embeddings.npy
└── paper_faiss.index
```

---

## Note

The following files are intentionally excluded from GitHub:

- arxiv-metadata-oai-snapshot.json
- ai_ml_papers.csv
- paper_embeddings.npy
- paper_faiss.index

They can be regenerated by following the above steps.
